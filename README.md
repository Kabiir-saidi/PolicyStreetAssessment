Steps to run
1. Export PolicyStreetAssessment.bacpac into SQL Express Server
2. Go to appsettings.json
3. point ConnectionString -> DefaultConnection to your SQL Express and Database name e.g : "DefaultConnection": "Server=localhost\\SQLEXPRESS01;Database=PolicyStreetAssessment;Trusted_Connection=True;TrustServerCertificate=True;MultipleActiveResultSets=True;",

API Swagger : https://localhost:44386/swagger/index.html
